#115 Humidity Brick

Overview

湿度センサーモジュールを使用したBrickです。

温度、湿度の情報を取得することができます。

接続

アナログコネクタ(A0〜A5)のいずれかに接続します。

Parts Specification

Document
DHT11

回路図

ソースコード

A0コネクタに接続して、湿度を計測するサンプルになります。

このサンプルコードでは外部ライブラリを使用します。

  • GitHub Library

ライブラリ名：「DHT sensor library」

#include "DHT.h"
DHT dht(A0, DHT11);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {

  delay(1000);

  float h = dht.readHumidity();
  float t = dht.readTemperature();

  Serial.print("Hum: ");
  Serial.print(h);
  Serial.print(" %");
  Serial.print("  Temp: ");
  Serial.print(t);
  Serial.println(" *C");
}
